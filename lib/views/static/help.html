Help

Using Trinket

Creating a Trinket

Click "New Trinket" from the dropdown menu in the navigation bar and select a language. You can write code in the editor and click "Run" to see the output.

Saving Your Work

Sign up for an account to save your trinkets. Once logged in, your work is automatically saved.

Sharing Trinkets

Each trinket has a unique URL you can share. You can also embed trinkets in other websites using the embed code.

Creating Courses

Trinket allows educators to create interactive coding courses with embedded trinkets, text, and media.

  1. Go to your library and click "New Course"
  2. Add topics and pages to organize your content
  3. Embed trinkets for interactive code examples
  4. Share your course URL with students

Self-Hosting & Development

Trinket is open source software. For information on running your own instance, see the GitHub repository.

Support

For bugs and feature requests, please open an issue on GitHub.
