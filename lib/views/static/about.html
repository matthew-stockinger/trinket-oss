About Trinket

Trinket is an open source browser-based coding platform that lets you write and run code without installing anything.

Originally created by Elliott Hauser and Brian Marks, Trinket was built to make coding accessible to learners everywhere. Whether you're a student writing your first Python program or an educator creating interactive lessons, Trinket provides the tools you need.

Features

  • Write and run Python, HTML, Java, and more in your browser
  • Create and share interactive coding projects
  • Build courses with embedded code examples
  • No downloads or installation required

Open Source

Trinket is released under the CC0 1.0 license. You can run your own instance, contribute to development, or use the code however you like.
